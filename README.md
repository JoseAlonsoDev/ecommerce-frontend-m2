# Pokochos TCG

> Sitio estático de ejemplo para una tienda de cartas coleccionables (Pokémon, Magic, One Piece).

**Estado:** Proyecto estático (HTML/CSS/JS) — listo para servir en un servidor estático o abrir localmente.

**Objetivo:** Sitio de catálogo con carrito simple usando LocalStorage y Bootstrap.

**Características principales**
- Página principal con carrusel (banner) y grilla con tres productos.
- Carrito de compras persistente en `localStorage` con badge de cantidad.
- Notificación tipo toast al añadir productos al carrito.
- Diseño responsivo con Bootstrap.

**Tecnologías**
- HTML5
- CSS (Bootstrap + `ASSETS/CSS/styles.css`)
- JavaScript (vanilla) — `ASSETS/JS/script.js`

**Requisitos**
- Navegador moderno (Chrome, Edge, Firefox).


**Estructura del proyecto**
- `index.html` — Página principal con banner y productos.
- `contact.html` — Página de contacto.
- `cart.html` — Vista del carrito y resumen de compra.
- `ASSETS/CSS/bootstrap.min.css` — Bootstrap.
- `ASSETS/CSS/styles.css` — Estilos personalizados.
- `ASSETS/IMAGES/` — Recursos de imagen (banner, productos).
- `ASSETS/JS/bootstrap.bundle.js` — Bootstrap JS (incluye Popper).
- `ASSETS/JS/script.js` — Lógica del carrito, badge y toasts.
